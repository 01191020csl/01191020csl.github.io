我的码风

  • #include<bits/stdc++.h> using namespace std;
  • 计数题#define mod 998244353 #define ll long long，且夹在#include和using之间；其他地方从不#define
  • 数组开在很多行，并且看起来比较相关的开在一起
  • 变量在使用前定义
  • 逗号前无空格，逗号后有空格
  • 大括号左括号前一个空格不换行，右括号单独占一行
  • 严格缩进
  • 一般运算符前后都带空格（中括号内不带）
  • 较长的表达式运算符前后都不带空格（赋值不算）
  • 不写奇怪的表达式
  • 极长的表达式分行写（新行必定运算符开头且均缩进一个tab），行长平均（包括最后一行）
  • 数据结构每个节点开一个结构体（节点处只有一个信息的除外），并且不冲突时必有结构体名为node，数组名为nd
